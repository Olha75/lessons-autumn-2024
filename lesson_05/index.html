Властивості блоків, відступи між елементами margin та padding

Візуально однаковові блоки (крім наповнення контентом), але різні записи
також прибрані фантомні відступи

  1. padding: 15px margin: 15px
  2. padding: 15px 15px margin: 15px 15px
  3. padding: 15px 15px 15px margin: 15px 15px 15px
  4. padding: 15px 15px 15px 15px margin: 15px 15px 15px 15px
  5. padding-inline:15px; padding-block:15px; margin-inline:15px; margin-block:15px;
  6. padding-top: 15px; padding-bottom: 15px; padding-left: 15px; padding-right: 15px; margin-top: 15px; margin-bottom: 15px; margin-left: 15px; margin-right: 15px;
  7. padding-block-start: 15px; padding-block-end: 15px; padding-inline-start: 15px; padding-inline-end: 15px; margin-inline-end: 15px; margin-inline-start: 15px; margin-block-end: 15px; margin-block-start: 15px;
  8. або мікс, наприклад: padding-block-start: 15px; padding-inline: 15px; padding-bottom: 15px; margin-inline-end: 15px; margin-left: 15px; margin-block: 15px;

Поведінка блоків при використанні різних стилів по ширині та висоті

  • max-with: 200px або max-inline-size
  • min-with: 200px або min-inline-size:200px
  • with: 200px або inline-size:200px
  • max-height: 100px або max-block-size:100px
  • min-height: 100px або min-block-size:100px
  • height: 100px або block-size:100px
